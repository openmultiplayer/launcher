## 🟣 Usage

> Use the **open.mp Launcher** to browse a live, populated server list and find any server you want to play on.
> Just download it from the [Releases](https://github.com/openmultiplayer/launcher/releases/latest) page and run it!

## 💜 Development

<details>
<summary><b>Prerequisites</b></summary>
<br>

- 🦀 [Rust nightly toolchain](https://rust-lang.github.io/rustup/concepts/channels.html)
- 🟩 [Node.js + npm](https://nodejs.org/en/download) — **any version except v20.6**

</details>

<details open>
<summary><b>Getting Started</b></summary>
<br>

**1. Clone the repository**
```bash
git clone https://github.com/openmultiplayer/launcher
```

**2. Install dependencies & start**
```bash
cd launcher
yarn        # or npm install
yarn start  # or npm start
```

**3. Build a release version**
```bash
yarn tauri build
```

</details>
